# Changelog

All notable changes to this project are documented in this file.

The format is based on Keep a Changelog, and this project uses semantic versioning.

## [1.0.0] - 2026-05-16

### Added
- Initial static accessibility crawler.
- WCAG 2.1 AA rule pack.
- Section 508 rule pack.
- Optional WCAG 2.2 enhanced rule pack.
- Selenium-based capture design for Edge, Chrome, and Edge IE-mode-assisted workflows.
- Edge IE-mode-assisted manual validation workflow and disclaimer handling.
- HTML, Markdown, CSV, JSON, and executive summary reports.
- Azure DevOps backlog export.
- PDF guidance overlay extraction.
- Portable ZIP release packaging scripts and GitHub Actions release workflow.
