# Manual Testing Checklist

Use this checklist after static crawling:

- Confirm every page and workflow is in scope and authorized for testing.
- Test keyboard-only operation for navigation, forms, dialogs, menus, custom controls, and error recovery.
- Confirm visible focus indication and focus order.
- Test forms with required fields, errors, validation messages, and successful submission.
- Test representative pages with screen readers.
- Review color contrast where computed style evidence is incomplete.
- Validate dynamic content, modal dialogs, accordions, tabs, and live regions.
- Review PDF, media, and downloadable content separately.
- For IE-mode systems, validate legacy controls manually.
