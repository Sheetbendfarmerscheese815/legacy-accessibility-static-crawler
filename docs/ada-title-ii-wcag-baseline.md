# ADA Title II / WCAG Baseline

This project uses WCAG 2.1 Level AA as the default static rule baseline for ADA Title II web and mobile app accessibility planning.

Important positioning:

- The crawler assists accessibility assessment and remediation planning.
- It does not certify ADA compliance.
- Static checks cannot prove full WCAG conformance.
- Manual accessibility testing, keyboard testing, screen reader testing, user-flow validation, and assistive technology testing remain required.

Reference: https://www.ada.gov/resources/2024-03-08-web-rule/
