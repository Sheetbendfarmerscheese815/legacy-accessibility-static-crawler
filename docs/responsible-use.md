# Responsible Use

Use this crawler only on systems you are authorized to assess.

Security and privacy defaults:

- Same-domain crawling is enabled by default.
- Query strings can be redacted.
- Credentials are not stored.
- Cookies and browser profiles are not stored by default.
- Reports, screenshots, captures, raw HTML, and cookies are gitignored.
- Manual login is supported for protected systems without saving credentials.

Review generated captures before sharing reports. HTML and screenshots can contain sensitive content.
